# Suggested Commands – WhoCallMe

## Project Generation (MUST do before opening Xcode)
```bash
# Install Tuist (first time / CI)
mise install tuist

# Install SPM dependencies
mise x -- tuist install

# Generate .xcodeproj / .xcworkspace files
mise x -- tuist generate
```
Never edit .xcodeproj files directly — they are generated by Tuist.

## Build
```bash
xcodebuild -workspace WhoCallMe.xcworkspace -scheme App -configuration Debug build
xcodebuild -workspace WhoCallMe.xcworkspace -scheme App -configuration Release build
```

## Test
```bash
# All tests
xcodebuild -workspace WhoCallMe.xcworkspace -scheme App \
  -destination 'platform=iOS Simulator,name=iPhone 16' test

# Single test class
xcodebuild -workspace WhoCallMe.xcworkspace -scheme App \
  -destination 'platform=iOS Simulator,name=iPhone 16' \
  -only-testing:AppTests/WhoCallMeTests test
```

## Fastlane (CI/CD)
```bash
# Deploy to TestFlight
fastlane ios release description:"변경사항" isReleasing:false

# Submit to App Store review
fastlane ios release description:"변경사항" isReleasing:true
```

## Mise (tool version manager)
```bash
mise install          # Install all tools in .mise.toml
mise x -- <command>   # Run a command with mise-managed tools
mise trust            # Trust .mise.toml if prompted
```
