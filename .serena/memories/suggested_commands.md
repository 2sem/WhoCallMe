# Suggested Commands – WhoCallMe

## Project Generation (MUST do before opening Xcode)
```bash
# Install SPM dependencies (run before generate)
mise x -- tuist install

# Generate .xcodeproj / .xcworkspace files
mise x -- tuist generate --no-open
```
Never edit .xcodeproj files directly — they are generated by Tuist.
After any Project.swift change, re-run tuist install + generate.

## Build
```bash
# Debug build for simulator
mise x -- tuist xcodebuild build -scheme App -configuration Debug -destination 'generic/platform=iOS Simulator'
```

## Test
```bash
xcodebuild -workspace WhoCallMe.xcworkspace -scheme App \
  -destination 'platform=iOS Simulator,name=iPhone 16' test
```

## Fastlane (CI/CD)
```bash
# Setup certs + version bump
fastlane ios setup

# Deploy to TestFlight
fastlane ios upload description:"변경사항" isReleasing:false

# Submit to App Store review
fastlane ios upload description:"변경사항" isReleasing:true
```
Note: `ios release` lane was renamed to `ios upload` in ci/upgrade-deploy (PR #5).

## Git
```bash
git checkout -b feat/my-feature   # Always branch from main first
git status
git log --oneline -10
```

## Mise (tool version manager)
```bash
mise install          # Install all tools in .mise.toml
mise x -- <command>   # Run a command with mise-managed tools
```
